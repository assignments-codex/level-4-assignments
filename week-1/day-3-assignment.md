#### Objective

Build a small Python script that uses variables, functions, and loops to solve a simple problem.

#### Requirements

1. Create a Python file that runs from the terminal.
2. Use at least one list or dictionary.
3. Use at least two functions.
4. Use at least one loop.
5. Print a clear result to the terminal.
6. Include a short README that explains:
   - what the script does
   - how to run it
