#### Goal

Add delete functionality and keep the interface in sync.

#### Steps

1. Complete what we built in lecture.
2. Add a delete action to one record.
3. Send the delete request through Flask.
4. Remove the deleted record from the UI.
5. Be ready to explain how your app stays in sync after an update or delete.

#### Extra practice if you want

- Add a confirm message before delete.
- Show a success message after delete.
- Add a simple refresh option.
