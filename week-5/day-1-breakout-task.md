#### Goal

Build the read side of a full stack app.

#### Steps

1. Complete what we built in lecture.
2. Display a list of records in React.
3. Add one detail view, selected view, or expanded card view.
4. Keep the layout clear and easy to use.
5. Be ready to explain how the data gets from the database into the page.

#### Extra practice if you want

- Add a loading state.
- Add an empty state message.
- Add a second component for list items.
