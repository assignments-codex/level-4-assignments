### Capstone interview rubric (100 pts)

A. Live coding (50 pts)
- Correct, runnable solution
- Handles an edge case or input validation
- Clear, idiomatic code
- Think-aloud reasoning
- Self-check and revise

B. Explaining code (30 pts)
- Design intent and trade-offs
- Data flow and error handling
- Constraints and alternatives

C. Presentation (20 pts)
- Live link first; clear story
- Accurate vocabulary; time managed
- Q and A handled precisely
- Smooth transitions
