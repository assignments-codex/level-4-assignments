#### Objective

Build a small full stack feature with React and Flask.

#### Requirements

1. Use React for the interface.
2. Use Flask for the backend.
3. Fetch data from Flask and display it in React.
4. Send form data from React to Flask.
5. Show the result of the submit in the interface.
6. Include a short README that explains:
   - what the app does
   - how to run the frontend
   - how to run the backend

#### Submission

GitHub repo link

#### Rubric (20 pts)

- Meets the objective 0-5
- React to Flask connection works correctly 0-5
- Form submission works correctly 0-5
- Code is organized and readable 0-5
