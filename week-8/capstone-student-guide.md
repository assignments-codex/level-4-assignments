# Level 4 Capstone - Student Guide

## Objective

Build and present a small full stack app that shows what you learned in Level 4.

## What you are building

Your project should include:

- a React front end
- a Flask back end
- a database
- at least one clear user flow that works from front to back
- deployment for the back end and front end

Keep the scope small. A simple app that works well is better than a large app that is unfinished.

## Minimum requirements

- public GitHub repo
- live backend URL
- live frontend URL
- at least one database-backed feature
- at least one route in Flask that reads or writes real data
- at least one React page that uses your backend
- at least one form or action that changes data
- README with setup steps and a short project description

## Presentation

Be ready to:

1. explain what your app does
2. demo the main user flow
3. show the main frontend and backend files
4. explain how the frontend, backend, and database work together

## What to submit

- GitHub repo link
- live frontend URL
- live backend URL
- README with:
  - project description
  - setup steps
  - how to use the app
  - where the frontend and backend are hosted

## Tips

- Keep the design simple.
- Focus on making the main feature work.
- Make sure you understand your own code.
- Build a project you can explain clearly.
