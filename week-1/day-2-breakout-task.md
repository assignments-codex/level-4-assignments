#### Goal

Practice conditionals, loops, and functions in Python.

#### Steps

1. Complete what we built in lecture.
2. Write at least one `if` statement.
3. Use a loop to work through a list of values.
4. Create at least two functions and call them.
5. Be ready to explain when you would use a loop and when you would use a function.

#### Extra practice if you want

- Add a second condition with `elif`.
- Return a value from a function.
- Use a loop to total numbers or count items.
