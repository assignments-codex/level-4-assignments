#### Goal

Work with files, JSON, modules, and one installed Python package.

#### Steps

1. Complete what we built in lecture.
2. Read from a text file or JSON file in Python.
3. Write or update data in a file.
4. Move one function into a second Python file and import it.
5. Install one package with `pip` and use it in a simple way.

#### Extra practice if you want

- Store a list of objects in JSON.
- Split your project into two or three small files.
- Add basic user input to choose an action.
