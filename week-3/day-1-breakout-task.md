#### Goal

Connect a React app to Flask and display backend data in the interface.

#### Steps

1. Complete what we built in lecture.
2. Create or use a React app for this week.
3. Fetch data from a Flask route.
4. Render the returned data on the page.
5. Be ready to explain the difference between the React code and the Flask code.

#### Extra practice if you want

- Add a loading message.
- Add an empty state message.
- Display the data in cards or a list.
