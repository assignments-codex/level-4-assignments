#### Goal

Organize a full stack app into clear frontend and backend pieces.

#### Steps

1. Complete what we built in lecture.
2. Break your React app into useful components.
3. Keep your Flask routes focused on one job.
4. Make the main user flow easier to follow.
5. Be ready to explain which files handle the frontend, backend, and data.

#### Extra practice if you want

- Add a reusable form component.
- Move repeated backend code into a helper.
- Improve the layout of the main page.
