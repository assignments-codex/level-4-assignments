#### Objective

Build a small Flask API with multiple working routes.

#### Requirements

1. Create a Flask app with at least three routes.
2. Include at least one GET route that returns JSON.
3. Include at least one POST route.
4. Use either a route parameter or a query parameter.
5. Test the routes with Postman.
6. Include a short README that explains:
   - what routes the API has
   - how to run the server
   - how to test one route

#### Submission

GitHub repo link

#### Rubric (20 pts)

- Meets the objective 0-5
- Routes work correctly 0-5
- Request and response patterns are used correctly 0-5
- Code is organized and readable 0-5
