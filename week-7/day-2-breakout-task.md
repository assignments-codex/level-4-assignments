#### Goal

Deploy a React frontend with S3 and CloudFront.

#### Steps

1. Complete what we built in lecture.
2. Build the React app for production.
3. Upload the build files to S3.
4. Connect CloudFront to the frontend.
5. Confirm the site loads from the live URL.

#### Extra practice if you want

- Update the app and redeploy it.
- Add one simple image asset.
- Check the difference between the bucket URL and the CloudFront URL.
