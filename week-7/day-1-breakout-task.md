#### Goal

Set up AWS safely and understand the pieces needed for frontend hosting.

#### Steps

1. Complete what we built in lecture.
2. Set a budget for your AWS work.
3. Review IAM basics and the permissions needed for this week.
4. Create or review the S3 bucket and CloudFront setup used in class.
5. Be ready to explain which part hosts the files and which part delivers them.

#### Extra practice if you want

- Review public vs private access.
- Write down the steps for deploying the frontend.
- Compare local hosting to cloud hosting.
