### Capstone student guide

You will present a project and complete a short live-coding task. Focus on technical quality and clear explanations.

What to prepare
- Live link(s) and public repo(s)
- README with quick start and a short architecture description
- One clear user flow that works end to end
- A short demo script

Interview flow
1. Live demo
2. Code walkthrough
3. Q and A
4. Live coding

Submission
- Repo link(s) and live link(s) in the LMS
