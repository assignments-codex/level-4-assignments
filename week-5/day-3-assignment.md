#### Objective

Build a full stack app that reads and creates data.

#### Requirements

1. Display stored data in React.
2. Add a form that creates a new record.
3. Send create requests through Flask.
4. Save the data in the database.
5. Show the new record in the app after it is created.
6. Include a short README that explains:
   - what the app does
   - how to run the frontend
   - how to run the backend
