#### Objective

Build a Flask API that reads from and creates data in a database.

#### Requirements

1. Connect Flask to the database service used in class.
2. Add one route that reads records.
3. Add one route that creates a new record.
4. Test both routes successfully.
5. Keep the data model simple and easy to explain.
6. Include a short README that explains:
   - what data your API stores
   - how to run the backend
   - what routes are working

#### Submission

GitHub repo link

#### Rubric (20 pts)

- Meets the objective 0-5
- Read route works correctly 0-5
- Create route works correctly 0-5
