#### Goal

Connect Flask to a database and return stored records.

#### Steps

1. Complete what we built in lecture.
2. Connect your Flask app to the database used in class.
3. Read records from one table or collection.
4. Return those records from a Flask route.
5. Be ready to explain where the data is coming from.

#### Extra practice if you want

- Return only selected fields.
- Add ordering to the returned data.
- Create a second read route.
