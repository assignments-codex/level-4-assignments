#### Goal

Add update functionality to a full stack app.

#### Steps

1. Complete what we built in lecture.
2. Choose one record to edit in the interface.
3. Build a form or edit flow for that record.
4. Send the update through Flask to the database.
5. Show the updated value in the app.

#### Extra practice if you want

- Reuse the create form for editing.
- Add a cancel button.
- Show which record is currently being edited.
