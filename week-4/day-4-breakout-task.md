#### Goal

Connect your React app to database-backed Flask routes.

#### Steps

1. Complete what we built in lecture.
2. Fetch real stored data through Flask and render it in React.
3. Send a create request from React to Flask.
4. Show the new data in the interface.
5. Be ready to explain the full path from React to Flask to the database and back.

#### Extra practice if you want

- Add a loading message.
- Add a simple success message.
- Add one more component to display the data.
